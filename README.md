# CrowdStrike DevOps Failure Analysis

SEAD Case Study Project analyzing the CrowdStrike 2024 global outage.

## Project Components
- Case Study Report
- Presentation Slides
- DevOps Risk Dataset
- Machine Learning Implementation (Colab Notebook)

## Technologies Used
Python  
Pandas  
Scikit-learn  
Jupyter Notebook  

## Case Study Summary
The CrowdStrike outage of July 2024 occurred due to a faulty kernel-level update that caused Windows systems to crash globally.

This project analyzes:
- Agile governance failures
- DevOps CI/CD pipeline issues
- Risk management strategies
- Deployment safety improvements
